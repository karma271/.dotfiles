# .dotfiles

# Set up:

- Download this repo to the home directory:

```
git clone git@github.com:karma271/.dotfiles.git ~
```

- Download stow:

  - Mac

  ```
  brew install stow
  ```

  - Fedora:

  ```
  sudo dnf install stow -y
  ```

- cd nto `~/.dotfiles` directory and run:

```
stow .
```

or

```
# stow <folder>
stopw nvim
```
